# Mastermind
The Mastermind also known in Slovakia as Logik is a well-known puzzle game for two players. Where one player creates a secret combination of five pegs of different colors, and the other player tries to guess it. According to the agreement, the colors may or may not be repeated. After each attempt, the guessing player learns how many pins he placed correctly and how many incorrectly. 

## The Mastermind console game made in Java.
This game was created in Java programming language as a school assignment of the Technical University in Košice.
